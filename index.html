Customizable UI Features

Customizable UI features allow users to personalize their experience on a website or application, tailoring the interface to their needs and preferences. These features improve accessibility, user satisfaction, and overall usability. Specifically:

  1. Dark Mode: This feature provides a darker color scheme for the UI, which can be easier on the eyes, especially in low-light environments. Many users prefer dark mode for aesthetic reasons or to reduce eye strain during extended use, particularly in the evening.
  2. Font Size Toggle: Allowing users to adjust font size is essential for accessibility. People with visual impairments or those who find text difficult to read can increase the font size to suit their preferences. This ensures that the website or app is usable by a broader range of users.
  3. Animations Toggle: Some users may find animations distracting, overwhelming, or even triggering in certain cases (e.g., motion sensitivity or epilepsy). Offering a toggle to turn off animations lets users customize their experience, making the interface more comfortable for them.

On-Scroll Animation

On-scroll animations are important because they can significantly enhance user engagement by adding visual interest and dynamism to a website. They can draw attention to important elements, encourage exploration, and leave a positive impression on visitors.

Parallax Effect

Parallax is when things closer to you seem to move faster than things farther away. Imagine looking out a car windowâ€”nearby trees zoom by, but distant mountains barely seem to move. Websites use this idea to make scrolling feel more interactive by moving different layers on the page at different speeds.

Implementation Steps:

  1. Define a div container with a specific height for the parallax background in your HTML file.
  2. Apply CSS styles to create the parallax effect. The parallax scrolling effect is achieved using properties like background-attachment: fixed;. The other background properties are used to center and scale the image perfectly
    • background-attachment: fixed; ensures that the background stays fixed as the user scrolls.
    • min-height: 400px; sets a fixed height for the background container.
    • background-size: cover; makes sure the image covers the entire div.

Multi-Layer Parallax Effect

Multi-layer parallax scrolling involves multiple images moving at different speeds to enhance the depth effect. This is achieved using the perspective and transform properties in CSS.

This effect was originally used in early video games (like the classic Moon Patrol from 1982) to create a sense of depth using multiple layers moving at different speeds. In modern web design, it's used to create engaging, immersive experiences and guide users' attention through content in an interesting way.

Parallax graphics can hold a user's attention and improve engagement, you can use parallax scrolling to produce a one-page website where visitors can read an entire story about your product or brand without navigating to multiple pages.

Understanding the Code

Similar to the simple parallax scrolling, we must first define the structure in HTML and then apply CSS.

The key concepts that can be found in the github code:

  • Perspective - Defines how far objects appear in 3D space perspective: 10px;.
  • Layering - Background and foreground images are placed inside a container div2
  • Transformations - Each layer moves at different speeds using translateZ. Here background will move slower than foreground.
  • Preserve 3D - Ensures that layers remain in a 3D space using transform-style: preserve-3d;.
How it works:
  • The .wrapper sets up a scrolling area with a 3D perspective.
  • Inside .div2, two images (background and foreground) are placed.
  • The background is moved further back (translateZ(-10px) scale(2);) , making it appear slower.
  • The foreground is slightly closer (translateZ(-5px) scale(1.5);), making it move faster.
  • Both images are absolutely positioned to overlap and create the parallax effect.
